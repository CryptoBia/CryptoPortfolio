FAQ

Is it safe to enter my portfolio on this site?

As long as you don't share your personal portfolio URL your data should be safe. All portfolios are saved to the database in a completely anonymous fashion. This web app does not collect any cookies, does not have ads and is 100% open source. The source code can be accessed via Github (LINK!).

From the Cryptocampare API.

Why does this website have such a weird logo?

It is a celtic Triple Crescent Moon symbol. Triple moons symbolize change, just like the blockchain technology in general. And "moon" - as in "to the moon"...
